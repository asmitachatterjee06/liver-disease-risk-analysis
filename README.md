# Liver Disease Risk Analysis (Statistical Modeling)

## Overview
This project analyzes clinical data to identify key risk factors influencing liver disease using statistical modeling techniques.

## Objectives
- Identify significant biomarkers affecting liver disease
- Analyze relationships between clinical variables
- Apply regression models for inference

## Dataset
- Source: UCI Machine Learning Repository
- Size: 500+ patient records
- Features: Bilirubin, Albumin, Proteins, Enzymes

## Methodology
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Cramér’s V Test (categorical association)
- Chow Test (structural stability)
- Log-Normal & Gamma Regression Models

## Key Insights
- Certain biomarkers significantly impact liver disease risk
- Male patients show higher probability of liver dysfunction
- Strong relationships observed between biochemical markers

## Tools & Technologies
- R (dplyr, ggplot2, MASS)
- Statistical Modeling Techniques
